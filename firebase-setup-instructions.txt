PLEASE FOLLOW THESE STEPS TO ENABLE GOOGLE LOGIN:

1. Create a new file in this directory named ".env.local" (dot env dot local).
2. Copy the content below into that file.
3. Replace the placeholder values (e.g., "your_api_key_here") with your actual Firebase project configuration keys.
   - You can find these in the Firebase Console -> Project Settings -> General -> Your Apps -> SDK Setup and Configuration.

------------------ COPY BELOW THIS LINE ------------------
NEXT_PUBLIC_FIREBASE_API_KEY="your_api_key_here"
NEXT_PUBLIC_FIREBASE_AUTH_DOMAIN="your_project_id.firebaseapp.com"
NEXT_PUBLIC_FIREBASE_PROJECT_ID="your_project_id"
NEXT_PUBLIC_FIREBASE_STORAGE_BUCKET="your_project_id.appspot.com"
NEXT_PUBLIC_FIREBASE_MESSAGING_SENDER_ID="your_sender_id"
NEXT_PUBLIC_FIREBASE_APP_ID="your_app_id"
